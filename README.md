# E-Commerce Angular Project

A simple e-commerce app built with **Angular 20** and **JSON Server** as a fake backend. Uses **Bootstrap 5** with custom CSS for styling.

---

## Tech Stack

- Angular 20
- JSON Server (fake REST API)
- Bootstrap 5 + Custom CSS
- RxJS + Angular HttpClient
- Jasmine & Karma for testing

---

## Prerequisites

- Node.js & npm
- Angular CLI v20
- JSON Server (`npm install -g json-server`)

---
